🎨 novajs 动态 class / style 演示

下面 6 个 box 用不同语法绑定 class 和 style，状态由右上角控制实时变化

brightness: {{ brightness }}
① 对象语法
:class="{ on: on, off: !on, warn: warn }"
最常用。键是 class 名，值是布尔。
② 数组语法
:class="['base', on ? 'on' : 'off', warn ? 'warn' : '']"
支持混合字符串 + 条件。
③ 数组 + 对象混合
:class="['base', { on: on, warn: warn }]"
Vue 风格的灵活组合。
④ 与静态 class 合并
class="demo-box on" :class="{ warn: warn }"
静态 class 保留，动态 class 叠加。
⑤ 完整字符串
:class="statusText" (值: "{{ statusText }}")
直接赋值整个 class 字符串。
⑥ pill 徽章
{{ online ? '● 在线' : '● 离线' }} :class="['pill', online ? 'live' : 'dead']"

动态 style

⑦ 灯泡亮度（style 对象）
:style="{ background: ..., opacity: ..., boxShadow: ... }"
⑧ 进度条（style 字符串）
:style="'width: ' + brightness + '%; ...'"
⑨ 与静态 style 合并
style="padding: 20px; border: 2px solid #cbd5e1;" :style="{ borderColor: ... }"
静态 style 保留，动态 style 覆盖同名属性。
⑩ 旋转角度（动态 transform）
:style="{ transform: 'rotate(' + brightness * 3.6 + 'deg)' }"
